## Run Locally

Clonar el repositorio

```bash
git clone https://github.com/OscarFDiaz/react-journal-app.git
```

Ir al directorio del proyecto

```bash
cd my-project
```

Instalar dependencias

```bash
yarn install
```

Iniciar el servidor

```bash
yarn dev
```
